# AI4R Explore Agent Practical Assignment

This repository contains the source code, setup files, assets, and assignment handout for the AI for Autonomous Robotics exploration-agent reinforcement learning practical.

Start from the exercise guide:

- [Exploring_agent_DRL/README.md](Exploring_agent_DRL/README.md)
- [Assignment handout](AIAR_RL_practical_assignment_1-4-a.pdf)

The repository intentionally excludes trained checkpoints, Ray outputs, TensorBoard logs, local Conda environments, and Python build/cache artifacts. New training runs write checkpoints under `Exploring_agent_DRL/tmp/`, which stays local by default.
